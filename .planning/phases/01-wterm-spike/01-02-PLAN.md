  <how-to-verify>
**Start the spike app:**

1. Terminal 1 — Start Go backend:
   ```bash
   cd be && go run main.go
   ```
   Should see: "Listening on 127.0.0.1:8080"

2. Terminal 2 — Start React frontend:
   ```bash
   cd fe && npm run dev
   ```
   Should see: dev server at http://localhost:5173

3. Open browser to http://localhost:5173

**Validation Checklist — test each and note PASS/FAIL:**

- [ ] **V-01: Basic connection** — Enter SSH credentials (a server you have access to), click Connect. Terminal appears and shows shell prompt.
- [ ] **V-02: vim** — Run `vim`. Editor renders correctly. Type some text. Exit vim (`:q!`). No artifacts left on screen.
- [ ] **V-03: htop** — Run `htop`. Dashboard renders with colors, bars, and live updating. Press `q` to quit. Screen clears correctly.
- [ ] **V-04: tmux** — Run `tmux`. Session starts. Create split (`Ctrl+B %`). Navigate between panes. Detach (`Ctrl+B d`). Reattach (`tmux attach`). All rendering correct.
- [ ] **V-05: Resize** — Resize browser window. Terminal redraws correctly. Run `htop`, resize window — htop layout adjusts. Run `vim`, resize — vim redraws.
- [ ] **V-06: Copy/paste** — Select text in terminal with mouse (DOM rendering should support this). Paste into another app. Copy from another app and paste into terminal.
- [ ] **V-07: curses apps** — Run `mc` (Midnight Commander) or `nano`. Renders correctly with dialog boxes and menus.
- [ ] **V-08: Long output** — Run `dmesg` or `cat /var/log/syslog`. Scrollback works. No rendering corruption.

**After testing, report results as:**
```
V-01: PASS/FAIL — [notes]
V-02: PASS/FAIL — [notes]
...
V-08: PASS/FAIL — [notes]
```
