    ```typescript
    const getCwd = useCallback((): Promise<string> => {
      return new Promise((resolve, reject) => {
        const socket = wsRef.current
        if (!socket || socket.readyState !== WebSocket.OPEN) {
          reject(new Error('Not connected'))
          return
        }

        // Set up a one-time listener for the cwd response
        const originalOnMessage = socket.onmessage
        const timeout = setTimeout(() => {
          socket.onmessage = originalOnMessage
          reject(new Error('get-cwd timeout'))
        }, 5000)

        socket.onmessage = (event: MessageEvent) => {
          if (typeof event.data === 'string') {
            try {
              const msg = JSON.parse(event.data)
              if (msg.type === 'cwd') {
                clearTimeout(timeout)
                socket.onmessage = originalOnMessage
                resolve(msg.path || '')
                return
              }
            } catch {
              // Not JSON, ignore
            }
          }
          // Forward to original handler for non-cwd messages
          if (originalOnMessage) {
            originalOnMessage.call(socket, event)
          }
        }

        socket.send(JSON.stringify({ type: 'get-cwd' }))
      })
    }, [])
    ```

       **Simpler approach:** Add a `getCwdForSession` function that opens a temporary WebSocket to the backend, sends connect using the same credentials, execs pwd, and returns the path. But this creates a whole new SSH connection just for pwd — wasteful.

       **Best approach:** Use the existing WebSocket. Expose a session-to-websocket mapping. Since `useSSHSession` is per-TerminalPane instance and TabBar doesn't have access to those hooks, use a store-based approach:

       Actually, this is getting complex. The simplest approach: use `useSSHSession`'s `getCwd` but we need it accessible outside the hook. 

       **FINAL APPROACH (simplest):** Store WebSocket references in a global map so TabBar can access them.

       In `useSSHSession.ts`, add a module-level Map:
       ```typescript

         return wsMap.get(sessionId)
       }
       ```

       In the `connect` function, after creating the socket, register it:
       ```typescript
       wsMap.set(sessionId, socket)
       ```

       In the cleanup/onclose, remove it:
       ```typescript
       wsMap.delete(sessionId)
       ```

       In TabBar's `handleDuplicate`:
       ```typescript
       import { getSessionWebSocket } from '@/features/terminal/useSSHSession'

       const handleDuplicate = async () => {
         if (!activeSessionId) return
         const source = sessions.find(s => s.id === activeSessionId)
         if (!source || source.status !== 'connected') return

         // Get cwd from the active session's WebSocket
         const ws = getSessionWebSocket(activeSessionId)
         if (!ws || ws.readyState !== WebSocket.OPEN) return

         try {
           const cwd = await new Promise<string>((resolve, reject) => {
             const timeout = setTimeout(() => reject(new Error('timeout')), 5000)
             const handler = (event: MessageEvent) => {
               if (typeof event.data === 'string') {
                 try {
                   const msg = JSON.parse(event.data)
                   if (msg.type === 'cwd') {
                     clearTimeout(timeout)
                     ws.removeEventListener('message', handler)
                     resolve(msg.path || '')
                   }
                 } catch {}
               }
             }
             ws.addEventListener('message', handler)
             ws.send(JSON.stringify({ type: 'get-cwd' }))
           })

           duplicateSession(activeSessionId, cwd)
         } catch {
           // Fallback: duplicate without cwd (starts in home directory)
           duplicateSession(activeSessionId, '')
         }
       }
       ```
