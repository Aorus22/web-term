**PUT handler `UpdateSettings`:**
- Decode JSON body: `{ "settings": { "key1": "value1", ... } }`
- For each key-value pair: use GORM `db.Save(&db.Setting{Key: k, Value: v})` (upsert via primary key)
- Return JSON: `{ "settings": { ... updated map ... } }` (same format as GET, with defaults applied)
